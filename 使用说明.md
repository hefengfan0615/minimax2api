# MiniMax All-in-One 使用说明

## 功能概述

这是一个整合了所有MiniMax官网功能的Python单文件应用，包含：

1. ✅ **MiniMax Web Agent API 适配器
2. ✅ OpenAI兼容的API代理
3. ✅ 账号密码自动登录并获取token
4. ✅ 多账号管理和轮换
5. ✅ Web管理界面
6. ✅ 完整的FastAPI应用

## 快速开始


## API端点说明

### 自动登录
- **POST /api/minimax/login** - 使用MiniMax账号密码自动登录

### OpenAI兼容API
- **POST /v1/chat/completions** - 聊天完成
- **GET /v1/models** - 模型列表

### 管理API
- **GET /api/config** - 获取配置
- **POST /api/config** - 更新配置
- **GET /api/accounts/status** - 账号状态
- **POST /api/test** - 测试连接
- **POST /api/test-account/{idx}** - 测试指定账号
- **GET /api/usage** - 使用统计

### Web界面
- **/admin** - 管理界面
- **/health** - 健康检查
- **/** - 主页
